## Requirements

-   Python
-   Ffmpeg
-   A Unix OS

## Setting up

1. Install Python requirements:

```bash
pip install -Ur requirements.txt
```

2. Copy `example.env` to `.env` and add your credentials.

## Running

```bash
bash bridge.sh
```
